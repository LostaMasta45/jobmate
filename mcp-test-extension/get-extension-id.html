Cara 1: Dari chrome://extensions/

  1. Buka: chrome://extensions/
  2. Aktifkan "Developer mode" (kanan atas)
  3. Lihat di card extension "MCP Chrome Test Extension"
  4. Extension ID ada di bawah nama extension (format: abcdefghijklmnopqrstuvwxyz123456)

Cara 2: Dari folder

Extension ID di-generate dari path folder. Copy Extension ID dari chrome://extensions/ (cara 1 lebih mudah)

Setelah dapat Extension ID:

Jalankan command berikut di terminal (ganti YOUR_EXTENSION_ID dengan ID yang sebenarnya):

cd C:\Users\user\Music\JOBMATE notepad "C:\Users\user\AppData\Roaming\Google\Chrome\NativeMessagingHosts\com.chromemcp.nativehost.json"

Edit file tersebut, tambahkan extension ID Anda ke allowed_origins:

{ "name": "com.chromemcp.nativehost", "description": "Node.js Host for Browser Bridge Extension", "path": "C:\\Users\\user\\AppData\\Roaming\\npm\\node_modules\\mcp-chrome-bridge\\dist\\run_host.bat", "type": "stdio", "allowed_origins": [ "chrome-extension://hbdgbgagpkpjffpklnamcljpakneikee/", "chrome-extension://YOUR_EXTENSION_ID/" ] }
