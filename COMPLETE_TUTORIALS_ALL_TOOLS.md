## ğŸ­ STAR Method untuk Behavioral Questions

### What is STAR?
- **S**ituation - Setup the context
- **T**ask - Explain your responsibility
- **A**ction - Describe what you did
- **R**esult - Share the outcome (dengan metrics!)

### Example:

**Question**: "Tell me about a time you solved a difficult problem."

**Answer using STAR:**
```
Situation:
"Di perusahaan previous, kami facing masalah website load time 
sangat slow (8+ seconds) yang menyebabkan bounce rate 65%."

Task:
"Sebagai Frontend Developer, saya ditugaskan untuk identify dan 
solve performance issues dalam 2 minggu."

Action:
"Saya start dengan audit menggunakan Lighthouse dan GTmetrix. 
Saya found issues di image optimization, JavaScript bundles yang 
terlalu besar, dan unnecessary API calls. Saya implemented lazy 
loading, code splitting, image compression, dan optimize API queries."

Result:
"Hasilnya, load time turun dari 8 detik ke 1.8 detik (77.5% improvement). 
Bounce rate turun dari 65% ke 28%, dan conversion rate naik 40%. 
Management sangat satisfied dan solusi ini di-implement ke semua 
products lainnya."
```

ğŸ’¡ **Tips STAR Method:**
- **Specific** - Jangan vague, kasih detail
- **Quantify** - Include numbers always
- **Positive** - Focus on success, bukan failure
- **Relevant** - Connect ke posisi yang dilamar
